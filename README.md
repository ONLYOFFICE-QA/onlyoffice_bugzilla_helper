# OnlyofficeBugzillaHelper

Helper gem for working with bugzilla

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'onlyoffice_bugzilla_helper'
```

And then execute:

```shell script
bundle
```

Or install it yourself as:

```shell script
gem install onlyoffice_bugzilla_helper
```
